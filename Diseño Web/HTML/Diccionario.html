< head>
< head> en HTML se utiliza para proporcionar información esencial sobre la página y enlazar recursos externos que ayudan a dar forma y mejorar la experiencia del usuario y la optimización de motores de búsqueda. Los contenidos de < head> no se muestran directamente en la página pero son críticos para su funcionamiento y presentación. NOTA: LOS 'style' van sin puntos.
Baja
Por ejemplo, el head de este mismo documento tiene la siguiente forma: 
                            
<.head>
    <.meta charset="UTF-8">
    <.meta name="viewport" content="width=device-width, initial-scale=1.0">
    <.title>DICCIONARIO DE ETIQUETAS HTML
    <.style>
    /* Aplica estilos de centrado al contenedor principal */
        .container {

            justify-content: center;

Aquí podemos ver varias configuraciones hechas al código que no se ven reflejadas dentro de este documento, como puede ser el color de fondo, el color del codigo escrito, o incluso la disposición de los diferentes elementos de la tabla.
<.style>
Es una etiqueta que permite utilizar estilos de CSS dentro del propio codigo HTML, se crea dentro del Head y permite asignarle ciertas cualidades como colores, contornos, etc.
Baja
En el head de la pagina creamos los diferentes estilos que necesitamos para nuestra pagina: 
                            
<.style>
    .cuadrado {
    width: 200px; /* Ancho deseado */
    height: 200px; /* Altura deseada */
    text-align:left; /* Alinea el texto en el centro del cuadrado */
    }
<./style>
En este caso, este estilo encapsula el texto en cuadrados para que no supere cierto largo y ancho y se conserve la homogeneidad de la pagina. agregamos dentro de la etiqueta del elemento a editar la linea: "class="cuadrado"" para que esta tome los elementos y características del estilo dado.
<.link>
Es una etiqueta que permite enlazar una hoja de estilos CSS al fichero HTML, esto con el fin de poder manejar de manera separada, la estructura de la pagina y sus configuraciones gráficas.
Media
En el Head de la pagina creamos la siguiente linea de codigo 
                            
<.link rel="stylesheet" href="estilos.css">
Agregamos esta linea de codigo junto a la ubicación del fichero CSS en el que se encuentra la tabla de estilos CSS. en este Fichero deben encontrarse referenciados las diferentes modificaciones gráficas
<.script>
Es una etiqueta que permite enlazar un Script de Codigo comúnmente JavaScript al fichero HTML, esto con el fin de poder manejar de manera separada, la lógica de la Pagina y sus principales procesos internos.
Media
En el Head de la pagina creamos la siguiente linea de codigo 
                            
<.script src="script.js">
Remplazamos el "script.js" con la ubicación local del fichero js en el cual se encuentra la lógica del sistema
<.body>
es el cuerpo de la pagina, cuando no generas divisiones, todo lo que pongas dentro del body es el contenido que sera visible de la pagina.
Baja
dentro de cualquier estructura de html 5 se vera una pareja de llaves como estas: 
                            
<.body>
    CONTENIDO DE LA PAGINA
<./body>
si el codigo de la pagina no es generado dentro, este contenido no sera procesado como parte del campo visible. esto se hace para que los metadatos u otras configuraciones no se procesen como texto dentro de la pagina.
<.color>
Esta etiqueta es la encargada de cambiar todo lo relacionado con los colores de las cosas, desde textos, etc. al añadir colors como etiqueta te salen todas sus combinaciones asociadas como fondos, letras bordes, etc.
Media
Dentro de la etiqueta de apertura del elemento podemos agregar los colores buscados, también pueden ser agregados a un style dentro del documento o dentro de clases creadas en CSS: 
                        
<.etiqueta color: #ff0000;">
o solo color: #ff0000 cuando esta dentro de una clase
este color puede ser modelado por rgb, Hex o incluso links externos donde se encuentre el color deseado.
<.div>
es una etiqueta que divide en secciones las partes de la pagina, esto se hace con el fin de manejar diferentes bloques de codigo con diferentes configuraciones dentro de la misma pagina.
Media
dentro de un body en una estructura html 5 ponemos el siguiente par de etiquetas: 
                        
<.div>
    CONTENIDO DE LA DIVISION
< /div>
todo lo que este dentro del div hace parte de una partición diferente del codigo general de la pagina, esto permite generar diferencias, por ejemplo podemos crear clases CSS para diferentes elementos y aplicarlas por separado dentro de divisiones en el codigo, permitiendo la creación de banners, contenidos, animaciones, footers etc.
<.h1>
html al ser una sistema de etiquetas enfocado en la web, la jerarquía de los elementos es crucial para la correcta disposición de los elementos en la pagina, los "h" son una parte importante en esta labor, pues generan títulos organizados en la web.
Baja
dentro de un body en una estructura html 5 ponemos el siguiente par de etiquetas: 
                        
<.h1> TITULO DEL ELEMENTO < /h1>
los numeros involucrados dentro de la etiqueta van de 1 a 6 siendo 1 el orden superior y 6 el inferior, pueden entenderse como títulos, subtítulos y títulos terciarios respectivamente, este orden influye en la construcción de la pagina y su posicionamiento en la Web y se relaciona directamente con el SEO.
<.p>
Establece el contenido como párrafos dentro de la pagina
Baja
dentro de un body en una estructura html 5 ponemos el siguiente par de etiquetas: 
                        
<.p> 
    CONTENIDO DEL PÁRRAFO
<./p>
todo el contenido dentro de estas etiquetas sera tomado como texto perteneciente a un párrafo del texto de la pagina.
