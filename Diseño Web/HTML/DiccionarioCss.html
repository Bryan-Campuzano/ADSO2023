DICCIONARIO DE ETIQUETAS CSS

Etiqueta Descripción Dificultad Código
clase
es una colección de modificaciones estéticas encapsuladas bajo un nombre de clase
NOTA: TODAS LAS ETIQUETAS TIENEN UN ESPACIO INTERMEDIO O UN PUNTO
esto es para poder manejar el codigo html sin que el editor de texto compile el codigo introducido
Media
se establece la siguiente estructura dentro del fichero CSS 
                            
.NombreClase{

CONTENIDO DE LA CLASE

}
de esta manera podremos separar diferentes configuraciones estéticas o de funcionamiento dentro de una clase, es importante declarar las clases con un punto al inicio de la declaración, al momento de involucrarla dentro de un div o un body si que se invoca con el nombre sin punto.
color
establece el color de diversos elementos de la pagina
Baja
Se establece la siguiente linea en cualquier parte de la clase: 
                            
color: #FF0000; 

Cambia #FF0000 a tu color hexadecimal deseado
al añadir color al la edición, normalmente te salen todas las opciones que puedes editar el color. te salen todas sus combinaciones asociadas como fondos, letras bordes, etc.
font-family
Especifica la fuente del texto.
Baja
Se establece la siguiente linea en cualquier parte de la clase: 
                        
font-family: Arial, sans-serif;
establece en orden todas las fuentes permitidas dentro del documento, el orden establecido es el orden en el que se intenta aplicar la fuente, si por alguna razón alguna de las fuentes no esta instalada o no carga, se tienen las siguientes como opcion.
text-align
Alinea el texto dentro de un elemento.
Baja
Se establece la siguiente linea en cualquier parte de la clase: 
                        
text-align: center;
tiene preestablecidos ciertas posiciones dentro de un espacio delimitado, arriba, abajo, izquierda, derecha y centro. esto alinea el texto en esa determinada forma.
border
Define los bordes de un elemento.
Baja
Se establece la siguiente linea en cualquier parte de la clase: 
                        
border: 1px solid #000;
puedes definir el ancho y otras cualidades del borde dentro de sus parámetros.
padding
Define el espacio interior de un elemento.
Media
Se establece la siguiente linea en cualquier parte de la clase: 
                        
padding: 5px;
puedes establecer una especie de barrera imaginaria entre el borde externo del cuadro y el contenido real, como el borde interior de un marco.
top, right, bottom, left
Define la posición de un elemento
Media
Se establece la siguiente linea en cualquier parte de la clase: 
                        
top: 0; right: 0;
partiendo de una hipotética posición central, podemos mover en diferentes medidas, las posiciones cartesianas del elemento.
background-image
Define una imagen como fondo de la pagina
Media
Se establece la siguiente linea en cualquier parte de la clase: 
                        
background-image: url('imagen.jpg');
puede ser una imagen local, ubicando la ruta en la que se aloja o también un url externo. ademas si la imagen no cubre la totalidad de la pagina, esta imagen se repite como mosaico por default.
opacity
Define el nivel de transparencia de un elemento
Baja
Se establece la siguiente linea en cualquier parte de la clase: 
                        
opacity: 0.5;
esta linea define la opacidad al 50% lo que lo hace translucido con el fondo, mientras mas alto sea el nivel de transparencia, mas oscuro sera el objeto. y mientras menor sea, mas transparente.
